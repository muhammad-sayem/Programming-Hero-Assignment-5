Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents that represents the structure of an HTML or XML document as a tree of objects. It allows developers to access, modify, and interact with elements, attributes, and content on a webpage using languages like JavaScript.

How do you select an element from the DOM?

We can select in many ways:
1. document.getElementById('elementID')
2. document.getElementsByClassName('className');
3. document.getElementsByTagName('tagName');
4. document.querySelector('selector');
5. document.querySelectorAll('selector');

What is the event delegation in the context of DOM, and why is it useful?

Event delegation is a technique in the DOM where a single event listener is added to a parent element to manage events triggered by its child elements. Instead of adding multiple event listeners to individual child elements, the parent listens for events on its children and uses event bubbling to handle them.
Useful because:
Performance: It reduces the number of event listeners, especially when dealing with many child elements, improving performance and memory usage.
Dynamic content: It works for dynamically added child elements that weren't present when the event listener was initially set up.

How do you manipulate an element's attributes and styles using DOM?

We can manipulate an element's attributes and styles using the DOM in the following ways:
*const value = element.getAttribute('attributeName');
*element.setAttribute('attributeName', 'newValue');
*element.removeAttribute('attributeName');
